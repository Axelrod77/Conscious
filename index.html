Integration

3 lines of code.
Every field, voice-aware.

// 1. Install the SDK
npm install @conscious/react
// 2. Wrap your app
import { ConsciousProvider } from '@conscious/react'

<ConsciousProvider proxyUrl="/api/voice">
  <App />
</ConsciousProvider>
// 3. That's it. Every input and textarea
//    in your app is now voice-aware.
//    No other changes required.

<input type="text" />          // ✓ Voice-aware
<textarea />                  // ✓ Voice-aware
<input type="search" />        // ✓ Voice-aware

Auto-detects all inputs and textareas. Works with React 17+. Field-type aware — speaks "five" into a notes field, "5" into a number field.
